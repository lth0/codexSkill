# codexSkill

This repository synchronizes the locally installed Codex/Agents skills on this machine and generates an index.

## Usage

1. In Codex, mention a skill name such as `$nature-polishing`, or ask for a task matching that skill description.
2. Each skill directory is listed below. The entry point is normally `SKILL.md`.
3. `skills/codex` is copied from `C:\Users\User\.codex\skills`; `skills/agents` is copied from `C:\Users\User\.agents\skills`.
4. The sync script only adds or overwrites files. It does not automatically delete stale files that disappeared from the source directories.

## Manual Sync

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File scripts/sync-skills.ps1
```

Generate files without committing or pushing:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File scripts/sync-skills.ps1 -NoCommit -NoPush
```
